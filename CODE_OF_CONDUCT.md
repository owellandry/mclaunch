# Código de Conducta

## Compromiso

En MC Launch buscamos una comunidad técnica respetuosa, enfocada en construir software de calidad alrededor del launcher, el instalador y el mod del proyecto.

Toda participación en este repositorio debe priorizar:

- respeto entre personas
- comunicación clara y profesional
- colaboración constructiva
- seguridad y privacidad responsables

## Ámbito

Este Código de Conducta aplica a todas las interacciones relacionadas con el proyecto, incluyendo:

- issues, pull requests, discusiones y revisiones
- reportes de bugs y sugerencias
- comunicación técnica sobre `react-ui`, `electron`, `react-installer`, `mc-home-client` y CI

## Comportamiento esperado

Se espera que todas las personas participantes:

- traten a los demás con respeto, incluso en desacuerdos técnicos
- aporten feedback específico, útil y orientado a solución
- acepten críticas técnicas de buena fe
- eviten lenguaje hostil, despectivo o provocador
- reporten problemas de seguridad siguiendo la política en `SECURITY.md`

## Comportamiento inaceptable

No se tolera:

- acoso, insultos, ataques personales o discriminación
- amenazas, intimidación o publicación de datos personales
- spam, trolling o sabotaje deliberado del trabajo colaborativo
- difusión pública de vulnerabilidades activas sin un proceso responsable
- publicación o solicitud de secretos, credenciales o accesos no autorizados

## Guía para contribuciones

Para mantener un flujo sano y productivo:

- describe el contexto y los cambios de forma clara
- mantén discusiones centradas en el problema técnico
- evita discusiones personales
- respeta decisiones de mantenimiento y revisión

## Reporte de incidentes

Si detectas una conducta que incumpla este código:

1. Reúne contexto mínimo (enlace, fecha, resumen).
2. Reporta al mantenedor principal por GitHub.
3. Si el incidente involucra seguridad, usa el canal indicado en `SECURITY.md`.

Los reportes se tratarán con la mayor confidencialidad posible.

## Aplicación y medidas

El mantenimiento del proyecto puede tomar medidas según la gravedad y reincidencia, por ejemplo:

- advertencia privada
- limitación temporal de interacción
- bloqueo de participación en el repositorio

## Relación con la licencia

Este Código de Conducta regula la convivencia y colaboración en el proyecto.
No modifica ni reemplaza los términos de uso del software definidos en `LICENSE`.

## Aceptación

Al participar en este repositorio, aceptas cumplir este Código de Conducta.
